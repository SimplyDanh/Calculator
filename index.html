About Percentage & Math Calculator

A comprehensive PWA that combines percentage calculations, a full standard & scientific calculator with MathLive WYSIWYG input, 12 themes, and an audit tape all running client-side with zero dependencies on a server.

Tech Stack
HTML5 CSS3 Vanilla JS MathLive Math.js PWA
Security & Architecture
  • SRI hashes on CDN scripts
  • 100% client-side, zero data sent
  • Service Worker offline cache
  • localStorage state persistence
Features
  • 4 percentage calc types
  • Standard + Scientific modes
  • Memory (M+/MR/MC/M)
  • 12 themes + dark mode
  • Audit tape history
  • Keyboard shortcuts
Limitations
  • No server, DB, or cloud sync
  • Scientific mode needs internet
  • IEEE 754 float precision
  • No graphing or unit conversion
