Percentage & Math Calculator

v2.0 • Progressive Web App

What It Does

A comprehensive calculator that combines percentage calculation tools with a full-featured standard and scientific calculator. Includes real-time percentage computations, an interactive keypad with memory functions, a scientific mode powered by MathLive (WYSIWYG math input) and Math.js expression evaluation, and a full audit tape history.

Languages & Technologies

  • HTML5 — Semantic structure with accessibility markup (ARIA live regions, focus management)
  • CSS3 — 12 color themes (8 solid + 4 animated Aurora gradients), Neumorphic soft-UI buttons, Glassmorphism panels, responsive layout, dark mode
  • Vanilla JavaScript — No build tools or frameworks; all logic in a single file
  • MathLive — WYSIWYG math field with virtual keyboard (CDN)
  • Math.js — Safe expression evaluation for scientific formulas (CDN)

Scope

  • Percentage calculations: "X% of Y", "X is what % of Y", "X is Y% of what?"
  • Standard calculator: digits, arithmetic, memory (M+/MR/MC), percentage key
  • Scientific mode: unlimited math rows, real-time evaluation, copy results
  • State persistence via localStorage (theme, mode, cards, history)
  • Keyboard shortcuts for rapid entry
  • Responsive design (desktop two-panel & mobile drawer)

Limitations

  • Runs entirely client-side — no server, no database, no cloud sync
  • Scientific mode requires an internet connection (MathLive & Math.js loaded from CDN)
  • Calculation precision limited by JavaScript floating-point (IEEE 754 double)
  • No unit conversions, graphing, or symbolic algebra
  • Audit tape is stored in-memory only (cleared on page reload unless persisted)
